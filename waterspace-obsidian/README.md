# WATERSPACE FACTORY 🏭

**Obsidian-хранилище команды ФАБРИКА WATERSPACE.**

Этот vault синхронизируется с сервером автоматически. Любые изменения, которые агенты вносят в wiki — коммитятся и пушятся сюда.

## Структура

| Папка/файл | Назначение |
|---|---|
| `README.md` | Этот файл — обзор vault |
| `MEMORY.md` | Устойчивый контекст WATERSPACE |
| `issyk-kul-insights.md` | Инсайты клиентов от Гарри |
| `issyk-kul-open-questions.md` | Открытые вопросы по Иссык-Кулю |
| `projects/` | Описания проектов и направлений |
| `decisions/` | Журнал принятых решений |
| `references/` | Полезные ссылки, материалы |
| `feedback/` | Обратная связь, инсайты |
| `inbox/` | Входящие заметки |
| `_templates/` | Шаблоны для новых заметок |
| `_setup/` | Инструкции по настройке |

## Правила

- Все заметки — **Markdown**
- Используй `[[wikilinks]]` для связей между заметками
- После редактирования через Obsidian — изменения подтянутся на сервер через git pull при следующем авто-коммите агентов
